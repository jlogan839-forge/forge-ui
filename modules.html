The Core Kit is the foundation. Domain modules build on top of it — same tokens, same patterns, same accessibility standards. Supply chain, finance, CRM, HR, and more. All free, forever.

How modules work
Every module inherits the Core Kit's token system. Override one variable at the root level and every component across every module updates automatically.
