Why Forge

Built for the real world

Backend-agnostic
Two files, one script tag. Works in Django templates, Rails ERB, Laravel Blade, Spring Thymeleaf, or plain HTML.
WCAG 2.1 AA built in
Every component ships with correct ARIA roles, keyboard navigation, focus management, and live regions. Not an afterthought.
CSS token system
Every value is a custom property. Override one variable to re-brand the entire kit. Dark mode is automatic.
Modular by design
Core Kit is the foundation. SCM, Finance, CRM, HR modules layer on top — same tokens, same patterns, no duplication.
MIT licensed
Use it in commercial products. White-label it. Modify it. No attribution required. No lawyers.
Community-driven
Every component, every module, every fix — built in public. Submit a PR and your work ships to thousands of projects.

Start building today

Download the kit, open index.html, and you're building. No account, no API key, no invoice.
